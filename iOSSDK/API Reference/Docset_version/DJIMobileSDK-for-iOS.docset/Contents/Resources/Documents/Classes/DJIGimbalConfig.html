DJIGimbalConfig Class Reference

Inherits from NSObject
Declared in DJIGimbal.h

Overview

This class provides advanced configuration of the gimbal. By changing the configuration, user can adjust the responsivity of the gimbal. Currently, the advanced configuration is only supported by OSMO.

configType

Gimbal configuration type. There are three preset types: fast, medium and slow. When the configuration type to DJIGimbalUserConfigTypeCustom1 or DJIGimbalUserConfigTypeCustom2, user can change the configuration by calling methods related to smoothTracking.
