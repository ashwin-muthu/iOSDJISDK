DJICameraParameters Class Reference

Inherits from NSObject
Declared in DJICameraParameters.h

Overview

Some of the camera’s parameters have dynamic ranges. DJICameraParameters provides the interface to query what the valid range is for a parameter. Type-casting is requried to get the corresponding enumerator value for each range element.

A category of DJICameraParameters. It provides methods to add or remove listeners for valid range change.
