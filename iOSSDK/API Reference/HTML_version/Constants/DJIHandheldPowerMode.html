DJIHandheldPowerMode

Handheld Power Mode

Definition

Constants

DJIHandheldPowerModeAwake

The Handheld Power Mode is awake. For OSMO, when it is in this mode, all the components in DJIHandheld are accessible.

Declared In DJIHandheldController.h.

DJIHandheldPowerModeSleeping

The Handheld Power Mode is sleeping. The handheld controller keeps the WiFi connection to the Mobile device alive but most other components are off. The power consumption is low in this mode. For OSMO, when it is in this mode, only the DJIHandheldController, DJIAirLink and DJIBattery are accessible.

Declared In DJIHandheldController.h.

DJIHandheldPowerModePowerOff

The Handheld Power Mode is power-off. Once this mode is set the delegate will receive this mode until the handheld device is shut down completely.

Declared In DJIHandheldController.h.

DJIHandheldPowerModeUnknown

The Handheld Power Mode in unkown.

Declared In DJIHandheldController.h.
